**WholeLocal: A Distributed Hybrid Approach to the Design of Resilient and Sustainable Supply Chains**

Naga Lalitha Kumar Mudiganti  
Frontiers in Sustainability | Manuscript ID: 1820269 | Under Review 2026

## Overview

This repository provides the complete replication package for the Distributed Hybrid Supply Chain Model (DHSCM). It implements all core equations from the manuscript and extends the calibration to three metropolitan benchmarks:

| Benchmark | R (retail, $B) | J_base (jobs) | LVRR_base | m (jobs/$1M) |
|-----------|----------------|---------------|-----------|--------------|
| Stylized mid-sized metro | $10B | 20,000 | 0.13 | 7.0 |
| **New York-Newark-Jersey City MSA** | **$420B** | **220,000** | **0.11** | **8.2** |
| **Los Angeles-Long Beach-Anaheim MSA** | **$280B** | **148,000** | **0.14** | **7.5** |

> **Important:** All model outputs are directional structural projections under stated assumptions. They are not empirical measurements, predictive forecasts, or LCA-validated quantities. See manuscript Section 2.8 for explicit modeling assumptions.

## Core Model Equations

All equations are implemented in `src/dhscm_model.py`. Key relationships:

**Local Value Retention Ratio:**
```
LVRR(α) = LVRR_base + α × (ΔR / R)
```

**Labor Stabilization Threshold:**
```
α_min(θ) = (θ × J_base) / (m × ΔR × 10³)
```

**Overproduction Dampening Proxy** *(structural proxy only; see note below):*
```
WasteRed(α, η) = 100 × η × α
```

**Recovery Time Reduction:**
```
RecoveryRed(α, γ) = 100 × γ × α
```

> **WasteRed Note:** This proxy represents directional overproduction dampening via bullwhip compression. It does NOT represent end-of-life circular loop waste reduction, an LCA-derived quantity, or an empirical measurement.

## Parameter Ranges and Sensitivity

| Parameter | Range | Basis |
|-----------|-------|-------|
| η (dampening coefficient) | [0.3, 0.7] | Lee et al. (1997); Chen et al. (2000); Disney & Towill (2003): 20-50% variance reduction from lead-time compression |
| γ (recovery coefficient) | [0.5, 0.7] | Tang (2006); Ivanov (2021): 15-30% recovery advantage for distributed configurations |
| m (employment multiplier) | [6.0, 9.0] | BEA RIMS II Type II published ranges for wholesale trade (NAICS 42) |

## New Metro Calibration Results (Key Findings)

At α = 0.35 (DHSCM-35 scenario), under midpoint parameter assumptions:

| Metric | Stylized | New York MSA | Los Angeles MSA |
|--------|----------|--------------|-----------------|
| LVRR gain | 0.13 → 0.24 | 0.11 → 0.18 | 0.14 → 0.21 |
| Incremental employment | 7,350 jobs | ~205,000 jobs | ~144,000 jobs |
| Retained revenue | $1,050M | $29,750M | $19,250M |
| Overproduction proxy | 17.5% | 17.5% | 17.5% |
| Recovery reduction | 21.0% | 21.0% | 21.0% |

> The hybrid equilibrium range α ≈ 0.25–0.35 is structurally robust across all three metro calibrations. Absolute employment and retention magnitudes differ substantially by metro scale; percentage-based resilience metrics (WasteRed, recovery reduction) are scale-invariant by model construction.
